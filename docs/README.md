## Code Organization

The codebase is organized into logical modules:

### Core Systems
- `src/agent/core_systems/` - Brain, consciousness, learning, and health monitoring

### AI Systems
- `src/agent/ai_systems/` - LLM providers, reasoning, and multi-agent orchestration

### Video Processing
- `src/agent/video_processing/` - Video editing, playback, stitching, and style learning

### Tools
- `src/agent/tools/` - Audio, vision, transcription, and research utilities

### Engines
- `src/agent/engines/` - Core processing engines and pipelines

### CUDA
- `src/agent/cuda/` - High-performance GPU computation and kernel generation

### Security
- `src/agent/security/` - Defense, validation, and safety systems

### Specialized
- `src/agent/specialized/` - Domain-specific agents and editors
